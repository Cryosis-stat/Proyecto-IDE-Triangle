
A continuación, se escriben algunas consideraciones a tener en cuenta si se desea
utilizar el software:

El software fue hecho para ejecutarse en consola, en el caso de Windows la consola
es posible que se cierre al terminar de ejecutar el programa por lo que se recomienda
utilizar el comando 	cmd.exe /k 	a la hora de ejecutar el programa para que 
no se cierre la consola.

Para abrir el código fuente se puede utilizar cualquier editor de C

El programa se compiló con el compilador de gcc (MinGW.org GCC-6.3.0-1)  de la 
version 6.3.0. Se recomienda utilziar el mismo si se desea compilar.
